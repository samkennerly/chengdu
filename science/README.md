# science

The **science/data** folder contains example datasets.
Use these for class projects or find your own datasets.

The **science/projects** folder contains example projects.
Read these for class project ideas or create your own.

`files.py` contains functions for reading and saving files.
If you run it as a script, it will test itself by saving to
(and reading from) the `data/test` folder.

This folder also contains class notes as Jupyter notebooks.
Jupyter, VSCode, and GitHub can display these notebooks.
